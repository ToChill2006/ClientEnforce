# ClientEnforce SEO Editorial Cadence

This plan keeps high-intent pages fresh, improves query coverage, and protects rankings after launch.

## Update cadence

- Weekly: monitor Search Console clicks, impressions, and CTR for the top 10 commercial pages.
- Monthly: refresh the top 5 commercial pages and top 5 blog posts by traffic/conversions.
- Quarterly: expand comparison coverage and prune underperforming pages.

## Monthly refresh workflow

1. Export queries for each target URL from Google Search Console.
2. Identify intent drift (new query variants, weaker CTR, ranking drops).
3. Update titles/descriptions only when query intent has changed.
4. Improve sections where visitors drop or bounce quickly.
5. Add 2-4 contextual internal links from recently updated pages.
6. Revalidate schema/canonical output and request reindexing.

## Content quality guardrails

- Keep sections specific to real buyer decisions and implementation steps.
- Do not add fabricated reviews, ratings, customer counts, or claims.
- Keep headings and copy concise; avoid keyword stuffing.
- Ensure each updated page links back to the pillar page and one relevant comparison page.
